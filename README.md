# EconomySim — MD пакет для мультиагентной разработки

Этот пакет предназначен для запуска разработки EconomySim с нуля до production-ready уровня с помощью нескольких AI-агентов.

Главное условие: агенты **самостоятельно ставят задачи, пишут код, создают интерфейс, тестируют, проверяют и доводят проект до production**.

## Как использовать

1. Скопировать все файлы в корень репозитория.
2. Начать с `AGENTS.md`.
3. Затем открыть `docs/01_PROJECT_BRIEF.md`, `docs/02_MULTI_AGENT_PROCESS.md`, `docs/05_ROADMAP.md`.
4. Каждый агент работает по циклу: Observe → Decide → Plan → Implement → Verify → Report → Create Next Tasks.

## Части продукта

- `apps/web` — игровой frontend.
- `apps/api` — backend/API.
- `apps/worker` — тики симуляции.
- `apps/constructor` — отдельный конструктор.
- `packages/simulation-core` — ядро симуляции.
- `packages/domain` — доменные типы.
- `packages/db` — база, миграции, репозитории.
- `packages/ui` — дизайн-система.

## Принципы

- Максимальное приближение к реальному миру.
- Нет pay-to-win.
- Рынок рождается из поведения населения, компаний и государств.
- Любая фича должна иметь код, UI/API для проверки, тесты и документацию.
- Любая денежная операция должна быть транзакционной и аудируемой.
