# Phase 18: UX/UI Polish

## Цель

Сделать первый игровой экран EconomySim понятным для нового игрока без изменения правил симуляции: карта остается главным экраном, а интерфейс объясняет состояние мира, риски и следующий полезный шаг.

## Реализованный UX-слой

- Верхний HUD показывает капитал игрока, тик, дату и действие следующего тика.
- Ресурсная панель добавляет быстрый обзор капитала, складов, доставок, ресурсов, экологии и рисков.
- Карта получила переключаемые слои:
  - экономика;
  - логистика;
  - ресурсы;
  - война;
  - загрязнение;
  - инфраструктура.
- Карточки стран показывают стабильность, режим и количество городов.
- Карточки компаний используют статусы, баланс, репутацию и предупреждение о банкротстве.
- Панель первых шагов ведет игрока через выбор карты/страны, создание компании, ресурсный сигнал, производство, тик и новости.
- Панель обратной связи показывает изменение цены, прибыль/убыток, дефицит и логистический риск.

## Accessibility

- Добавлен skip-link к карте.
- Интерактивная карта имеет `role="region"` и aria-label.
- Переключатели слоев используют `aria-pressed`.
- Ошибки API остаются заметным alert-паттерном, пустые состояния остаются читаемыми.
- Добавлены единые `focus-visible` outline для keyboard navigation.

## Границы фазы

Фаза не меняет экономические правила и не добавляет новые команды покупки ресурса или производства вручную. UI использует существующие данные backend/simulation-core и показывает доступные сигналы мира.
