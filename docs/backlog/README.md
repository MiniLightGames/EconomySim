# Backlog

AI-агенты создают здесь задачи автоматически.

## Naming

```text
P1-TASK-YYYYMMDD-HHMM-short-name.md
P1-BUG-YYYYMMDD-HHMM-short-name.md
SIM-YYYYMMDD-HHMM-short-name.md
ADR-XXXX-short-name.md
```

## Priority

- P0: критический блокер, безопасность, потеря данных, деньги ломаются.
- P1: важная фича или серьёзный баг.
- P2: качество, UX, performance, метрики.
- P3: cleanup/docs/refactoring.
