# 01. Project Brief — EconomySim

## Видение

EconomySim — web-игра, экономическая песочница и исследовательская симуляция мира. Игрок управляет капиталом, компаниями, инвестициями и контрактами в живой мировой экономике.

## Цель

Создать реалистичную симуляцию, где спрос, цены, производство, логистика, миграция, кризисы, войны и политика возникают естественно из действий населения, компаний, банков, государств, NPC и игроков.

## Архитектурные части

1. **Frontend** — игровая карта, панели стран/городов/компаний/рынков, новости, контракты.
2. **Backend** — API, авторизация, БД, транзакции, аудит, права, связь с ядром.
3. **Simulation Core** — тики, экономика, население, производство, логистика, рынки, банки, войны.
4. **Constructor** — отдельное приложение для разработчиков: товары, ресурсы, цепочки, здания, законы, тестовые мини-симуляции.

## Игрок

Игрок — глобальный экономический оператор, а не физический персонаж. Он может открывать компании, покупать активы, заключать контракты, брать кредиты, инвестировать, влиять на политику через СМИ/лоббизм и участвовать в серой экономике через игровые механики.

## Первый прототип

Маленькая глубокая планета-модель:

- 5–10 стран;
- 20–50 городов;
- 10–20 ресурсов;
- 25–50 товаров;
- население когортами;
- NPC-компании;
- банки;
- логистика;
- магазины/контракты;
- биржи;
- web-карта;
- базовый конструктор.
