# 15. Production Checklist

## Product

- [ ] есть onboarding;
- [ ] игрок понимает первые действия;
- [ ] есть рабочий экономический цикл;
- [ ] нет pay-to-win;
- [ ] UI объясняет причины изменений.

## Engineering

- [ ] CI green;
- [ ] Docker images build;
- [ ] migrations tested;
- [ ] seed works;
- [ ] commands documented.

## Backend

- [ ] auth;
- [ ] RBAC;
- [ ] validation;
- [ ] audit;
- [ ] ledger;
- [ ] rate limits.

## Frontend

- [ ] карта работает;
- [ ] ключевые панели работают;
- [ ] loading/error/empty states;
- [ ] responsive;
- [ ] important actions confirm.

## Simulation

- [ ] tick stable;
- [ ] scenario regression;
- [ ] metrics;
- [ ] event log;
- [ ] snapshot/restore.

## Data

- [ ] backups;
- [ ] restore tested;
- [ ] indexes;
- [ ] partitioning plan for events/metrics.

## Security

- [ ] dependency scan;
- [ ] secret scan;
- [ ] anti-cheat alerts;
- [ ] financial race tests.

## Operations

- [ ] monitoring dashboard;
- [ ] alerts;
- [ ] release playbook;
- [ ] rollback plan;
- [ ] incident template.
